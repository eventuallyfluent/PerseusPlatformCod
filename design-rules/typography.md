# Typography

The original design uses a strict split between display atmosphere and practical UI readability.

## Intended Fonts

- Display: Cinzel for the current app default. It gives the sharp Perseus serif character without making every section feel over-ornate.
- Ceremonial accent: Cinzel Decorative may be used only for rare brand marks or carefully reviewed hero-level display details, not as the global heading face.
- UI and body: DM Sans.
- Metadata and technical labels: JetBrains Mono.

## Rules

- Display font belongs in hero headings, page-level moments, public card titles where it remains readable, instructor names, stage titles, and major section titles.
- UI/body font belongs in cards, badges, forms, filters, navigation, pricing, and lesson/player surfaces.
- Editorial section headings may use a split title: strong ivory/purple text first, then optional italic lavender-to-gold emphasis for the key phrase.
- Decorative display type must be rare. If a heading starts to feel theatrical, use regular Cinzel.
- Catalog card titles should be readable and compact, not hero-sized.
- Avoid negative tracking in dense UI. Keep letter spacing for uppercase labels and badges only.
- Long course names must clamp gracefully and must not force large empty card areas.
